# Calendar Quick Start Guide

## Getting Started

### 1. Access the Calendar
Navigate to `/calendar` in the AIT-CORE Suite Portal.

### 2. Create Your First Event

**Method 1: Using the Create Button**
1. Click the "Crear evento" button in the toolbar
2. Fill in the event details
3. Click "Guardar"

**Method 2: Click on Calendar**
1. Click on any date or time slot in the calendar
2. The event modal will open with the selected date/time pre-filled
3. Add event details and save

### 3. View Modes

Switch between different views using the dropdown in the toolbar:
- **Mes (Month)**: See all events for the month
- **Semana (Week)**: Detailed weekly view with hourly slots
- **Día (Day)**: Single day view with all events
- **Agenda (List)**: List view of all events

### 4. Navigate Dates

**Quick Navigation:**
- Click "Hoy" to jump to today
- Use ← and → arrows to go to previous/next period
- Click on dates in the mini calendar (sidebar)

**Mini Calendar:**
- Located in the left sidebar
- Shows dots on dates with events
- Click any date to jump to it
- Use arrows to change months

### 5. Manage Events

**Edit an Event:**
1. Click on any event in the calendar
2. Modify the details in the modal
3. Click "Guardar"

**Delete an Event:**
1. Click on the event
2. Click the "Eliminar" button in the modal
3. Confirm deletion

**Move an Event:**
- Simply drag and drop the event to a new date/time
- Changes save automatically

**Resize an Event:**
- Hover over the bottom edge of an event
- Drag to adjust the end time
- Changes save automatically

## Common Tasks

### Create an All-Day Event
1. Create a new event
2. Toggle the "Todo el día" switch
3. The time selectors will change to date-only

### Add Event Details

**Location:**
- Click the location field and type the address or place name

**Attendees:**
1. Type an email address in the attendees field
2. Click "Agregar" or press Enter
3. Repeat for multiple attendees
4. Click the X on any badge to remove an attendee

**Reminders:**
1. Click the reminders dropdown
2. Select when you want to be reminded
3. Add multiple reminders if needed
4. Click X to remove a reminder

**Description:**
- Use the description field for additional notes
- Supports multi-line text

### Create Recurring Events

1. Toggle the "Repetir evento" switch
2. Choose frequency (Daily/Weekly/Monthly/Yearly)
3. Set interval (every N days/weeks/etc.)
4. Optionally set an end date
5. Save the event

### Organize with Calendars

**View Calendars:**
- Check the "Mis calendarios" section in the sidebar
- Each calendar has a color indicator

**Show/Hide Calendars:**
- Click the checkbox next to any calendar name
- Events from hidden calendars won't be displayed

**Select Calendar for Event:**
- When creating/editing an event
- Use the "Calendario" dropdown to choose which calendar

### Search Events

1. Use the search box at the top of the sidebar
2. Type to search by title, description, or location
3. Results update in real-time
4. Clear the search to see all events again

## Tips & Tricks

### Keyboard Efficiency
- Press Tab to navigate between fields in the modal
- Press Enter in the attendee field to quickly add emails
- Press Escape to close the modal

### Visual Organization
- Use different calendars for different types of events (Work, Personal, etc.)
- Each calendar has a unique color for easy identification
- Toggle calendars on/off to focus on specific types of events

### Quick Event Creation
- Single click on a date in month view creates an all-day event
- Click on a time slot in week/day view creates a timed event
- The selected date/time is automatically filled in

### Viewing Upcoming Events
- Check the "Próximos eventos" section in the sidebar
- Shows the next 5 upcoming events
- Click on any event to view/edit details

### Best Practices

1. **Use Descriptive Titles**: Make it easy to identify events at a glance
2. **Add Locations**: Especially useful for meetings and appointments
3. **Set Reminders**: Don't miss important events
4. **Use Multiple Calendars**: Organize events by category
5. **Add Attendees**: Keep track of who's involved
6. **Regular Cleanup**: Delete old or cancelled events

## Troubleshooting

**Can't see an event:**
- Check that the calendar it belongs to is visible (checked in sidebar)
- Verify you're viewing the correct date range
- Try searching for the event name

**Drag and drop not working:**
- Make sure you click and hold on the event
- Try refreshing the page if the issue persists

**Event modal not opening:**
- Check your browser console for errors
- Try clicking directly on the event title

**Search not working:**
- Make sure you're searching in visible calendars
- Check spelling of search terms
