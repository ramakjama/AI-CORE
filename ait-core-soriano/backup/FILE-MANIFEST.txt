USAGE QUICK REFERENCE
===========================================

SETUP:
  ./setup-backups.sh

BACKUP:
  ./backup-all.sh              # Everything
  ./backup-postgres.sh         # Database only

RESTORE:
  ./restore-all.sh -a          # Everything
  ./restore-postgres.sh        # Database only

VERIFY:
  ./verify-backups.sh          # Check integrity
  ./test-restore.sh            # Test restore

MONITOR:
  ./backup-stats.sh            # Statistics
  tail -f /backup/cron.log     # Watch logs

SCHEDULING
===========================================

Recommended Frequencies:
  - Full backup:      Daily at 2 AM
  - PostgreSQL:       Every 6 hours
  - Redis:            Every 4 hours
  - MinIO:            Daily at 3 AM
  - Elasticsearch:    Daily at 4 AM
  - Verification:     Weekly
  - Restore test:     Monthly

===========================================

Local (minimum): 500GB
Local (recommended): 2TB

S3 (estimated for 100GB DB):
  - Active backups: ~200GB
  - 30-day retention: ~400GB
  - 90-day retention: ~600GB
  - Cost: ~$8-15/month
